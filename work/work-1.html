About

PresentText is a Google Chrome extension that allows the user to put highlighted text into focus on a page and dims the light on the rest of the page. This allows for easy, distraction-free reading or for an actual presentation, where focus is needed on only a small passage.
Currently, I've only built the core functionality of PresentText, which includes the on/off functionality, layering a dark cover over the page, and smooth transitions. I plan to bring multiple colors and opacity options to the extension soon.

Design and Implementation

I wanted the extension to be quick and easy to use, so that two simply clicks would start the extension. An icon is built into Chrome browser when the user installs the extension, and it opens a smaller window with presentation options. The user can then start a session.
A PresentText session is dynamic. Whenever the user highlights a text passage, the remaining window of the browser is darkened with a low-opacity black layover, so that the highlighted portion appears bright. The layover is implemented with basically four rectangles, covering left, right, top, and bottom of the screen except for the highlighted portion.
PresentText is built on JavaScript
