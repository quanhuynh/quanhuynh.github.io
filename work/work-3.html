When I first started programming, I started contributing to a project called the Daily Programmer. It was a series of online coding challenges at different levels for learning or just for fun. For me it was both. I continued working on my personal Daily Programmer portfolio, and all of my projects can be found on my Github.
The portfolio includes many things, from easy projects for me to get used to a new language, to a simulator or a game. I have also taken inspiration from many topics and expanded them into my own projects.
For example, the first image you saw above was my personal interactive Conway's Game of Life simulator. I had a lot of fun making and using it. It runs a Conway's Game of Life simulation but the user can determine which cell dies or lives at any given time.
