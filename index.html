Things I do
  • Tradebook

    Tradebook is a web application made for paper-trading, a form of stock market simulator. The project includes most of a stock market simulator's basic features, such as buying, selling, ask/bid price system, watchlisting, etc. The application includes a simple user account system, where each user can have multiple trade portfolios. Each portfolio has its own set of holdings, account values, and watchlist.

    The application was built on the MeteorJS framework and uses the Yahoo Finance API for data. Source code here .

  • Java Matrix Package

    The Java Matrix Package is an external .jar file featuring the Matrix class. This class creates Matrix objects from 2D arrays or single values. The class has 20+ methods for working with matrices, from basic operations like adding, multiplying, to finding inverses, determinants, and row reductions.

    The .jar file and documentation for methods are available here .

  • CodeSaver

    CodeSaver is a desktop application for storing code snippets away for future references.

    The application was made with Java Swing and the JDBC API to store data with MySQL. Source code here .
